# Documento de Referencias para el Módulo 0

## I. Consideraciones Metodológicas de Citación

Para la documentación de estudios y la elaboración de informes de política y académicos en el ámbito técnico-económico, se recomienda seleccionar el estilo de citación según el campo de investigación o el propósito específico del informe. Los estilos más adecuados para este tipo de análisis son generalmente **APA** o **Chicago**.
