# Guion Educativo: Hipertrofia (Tema 2.2)

**Personaje**: Entrenador Experto.
**Duración Estimada**: 2:45 minutos.

---

**(00:00 - 00:30)**
**LOCUTOR**: Todos quieren músculos más grandes. Pero, ¿cómo sucede realmente? No es magia, es adaptación. Hay tres caminos para llegar a Roma, o en este caso, a la Hipertrofia.

**(00:30 - 01:10)**
**LOCUTOR**: El camino principal, la autopista, se llama Tensión Mecánica. Es someter a la fibra muscular a una carga pesada y obligarla a generar fuerza. Si no hay tensión, no hay razón para crecer. Punto. Esta debe ser la base de tu pirámide.

**(01:10 - 01:45)**
**LOCUTOR**: Luego tenemos el camino secundario: Estrés Metabólico. Es el "pump", el bombeo, el ardor. Aunque no uses mucho peso, si llevas el músculo a la fatiga metabólica, activas señales de crecimiento potentes. Es excelente para aislar músculos sin destrozar las articulaciones.

**(01:45 - 02:20)**
**LOCUTOR**: Y finalmente, el Daño Muscular. Las famosas agujetas. Muchos creen que es el objetivo, pero cuidado: es un efecto secundario. Si destrozas el músculo tanto que no puedes caminar en tres días, no puedes volver a entrenar. Y la frecuencia es clave.

**(02:20 - Ends)**
**LOCUTOR**: ¿La receta ganadora? Mucha tensión mecánica, una dosis estratégica de estrés metabólico, y el daño muscular justo y necesario para que el cuerpo pueda repararlo y supercompensar.
